# 入力フォーム レイアウト改善案

## 現状の問題点
- 5つのセレクトボックスが縦に並び、高さを取りすぎている
- 「00」という値だけでは各フィールドの意味が分かりづらい
- 日付、時刻、睡眠時間の関係性が視覚的に不明確

## 提案1: グループ化 + コンパクトレイアウト（推奨）

### コンセプト
- 2つの明確なグループに分ける
- 画面下部に固定し、1-2行で収める
- ラベルを追加して分かりやすく

### レイアウト構造
```
┌─────────────────────────────────┐
│ 睡眠記録                        │
├─────────────────────────────────┤
│ 睡眠開始                        │
│ [日付 ▼] [時 ▼] [分 ▼]        │
│                                 │
│ 睡眠時間                        │
│ [時 ▼] [分 ▼]                  │
│                                 │
│      [記録する]                  │
└─────────────────────────────────┘
```

### メリット
- グループ化により意味が明確
- ラベル追加で分かりやすい
- 高さを抑えられる（約50%削減）

---

## 提案2: 1行コンパクトレイアウト

### コンセプト
- 画面下部に1行で全て配置
- 最小限のスペースで最大限の機能

### レイアウト構造
```
┌─────────────────────────────────┐
│ [日付▼] [時▼] [分▼] ~ [時▼] [分▼] [記録] │
└─────────────────────────────────┘
```

### メリット
- 高さを最小限に抑制
- 画面下部に固定されて常に見える

### デメリット
- ラベルがないため、最初は分かりづらい可能性
- 小さな画面では詰まって見える

---

## 提案3: カード型グループ化

### コンセプト
- 各グループをカードとして視覚的に分離
- アイコンやガイドテキストを追加

### レイアウト構造
```
┌─────────────────────────────────┐
│ ┌─── 睡眠開始 ───┐              │
│ │ 📅 日付: [▼]   │              │
│ │ ⏰ 時刻: [時▼] [分▼] │        │
│ └─────────────────┘              │
│                                 │
│ ┌─── 睡眠時間 ───┐              │
│ │ ⏱️  [時▼] [分▼] │             │
│ └─────────────────┘              │
│                                 │
│      [記録する]                  │
└─────────────────────────────────┘
```

### メリット
- 視覚的に分かりやすい
- グループの関係性が明確

### デメリット
- やや高さが増える可能性

---

## 提案4: 文脈的に自然な順序（推奨第2位）

### コンセプト
- 「いつ？どれくらい？」という自然な思考順序
- 日付を大きく、時間を小さく

### レイアウト構造
```
┌─────────────────────────────────┐
│ 睡眠開始日時                    │
│ [2025-01-06 (今日) ▼]           │
│ [14 ▼] : [30 ▼]                │
│                                 │
│ 睡眠時間                        │
│ [1 ▼] 時間 [30 ▼] 分           │
│                                 │
│      [記録する]                  │
└─────────────────────────────────┘
```

### メリット
- 思考の流れに沿っている
- 日付選択を優先的に大きく表示

---

## 推奨: 提案1（グループ化 + コンパクトレイアウト）

### 実装のポイント
1. **ラベル追加**: 各グループに明確なラベル
2. **2行レイアウト**: 開始日時（1行）、睡眠時間（1行）
3. **視覚的分離**: グループ間に軽い区切り線またはスペース
4. **プレースホルダー改善**: 各セレクトボックスにプレースホルダーを追加
5. **画面下部固定**: stickyで常に見える位置に配置

### CSS改善案
- `form-row`を2つのグループに分割
- 各グループに`.form-group`クラスを追加
- ラベルを`.form-label`として追加
- セレクトボックスを`.form-select`として統一

### 期待される効果
- 高さ: 約50%削減（5行 → 2-3行）
- 理解しやすさ: ラベル追加により大幅改善
- 操作性: グループ化により入力順序が明確
