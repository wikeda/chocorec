# ちょこRec 画面ワイヤー詳細

## メイン画面（記録＋グラフ）
- ヘッダー
  - タイトル: ちょこRec
  - 右上: 履歴 / 種目
- 期間タブ
  - 週次 / 月次
  - タブ切替でグラフとサマリが更新
- サマリ
  - 今週（今月）の合計負荷
  - 記録回数
- グラフ（積層横棒）
  - 週次: 高さ固定
  - 月次: 日数に応じて縦スクロール
- 記録フォーム（ボトム固定・2行）
  - 行1: 日付 / 種目
  - 行2: 回数 / セット / 重さ(任意) + 記録ボタン
  - 種目選択で前回値を自動補完
  - 記録後: 次のプルダウンを自動選択 + グラフ更新

## 履歴画面
- ヘッダー
  - 戻る
  - 右上: CSV保存
- 一覧
  - 日付ごとにグループ
  - 行表示: 種目名（色付き）/ 回数×セット / 重さ（あれば）
  - ラジオ選択で「編集」有効化
- 編集ボトムシート
  - 編集項目: 日付/種目/回数/セット/重さ
  - 保存 / 削除 / キャンセル

## 種目管理画面
- ヘッダー
  - 戻る
  - 右上: 追加
- 一覧
  - 色インジケータ + 種目名
  - 並び替え（↑↓）
  - 編集ボタン
- 追加/編集ボトムシート
  - 種目名（20字以内）
  - 色（カラーピッカー＋HEX）
  - 保存 / 削除（編集時のみ） / キャンセル
- 論理削除
  - 削除済みは新規記録の選択肢から除外
  - 既存記録は保持

## CSV保存
- 履歴画面の「CSV保存」ボタンで保存先選択
- 保存完了トースト＋保存先表示（可能なら）

## ホーム画面ウィジェット（中サイズ）
- 表示: 今週合計負荷 / 今週記録回数
- タップ: メイン画面の記録フォームへ遷移
