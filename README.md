# RecSleep

睡眠時間を記録するスマートフォン専用Webアプリケーション。

## 特徴

- 📱 スマートフォン専用に最適化されたUI
- 💤 昼寝を含む1日複数回の睡眠記録に対応
- 📊 週次・月次の積層グラフで睡眠パターンを可視化
- 💾 ローカルストレージでデータを保存（オフライン動作可能）
- 📥 CSV形式でデータをエクスポート可能
- ✏️ 過去の記録を編集可能

## 機能

### 睡眠記録
- 睡眠開始日時を選択（過去7日まで）
- 睡眠時間を5分単位で選択
- 22時以降～7時までの睡眠は自動的に「夜の睡眠」として分類
- それ以外は「昼の睡眠」として分類

### グラフ表示
- 週次グラフ：過去7日間の睡眠時間を積層グラフで表示
- 月次グラフ：過去30日間の睡眠時間を積層グラフで表示
- 夜の睡眠：濃い青色で表示
- 昼の睡眠：オレンジ色で表示

### 過去記録管理
- 過去の記録を一覧表示
- 記録をクリックして編集可能
- 日付ごとにグループ化して表示

### データエクスポート
- CSV形式で全記録をダウンロード
- Excel等で開ける形式

## 使用方法

1. `index.html`をブラウザで開く
2. フォームから睡眠開始日時と睡眠時間を選択
3. 「記録する」ボタンをクリック
4. グラフや表で記録を確認

## 技術スタック

- HTML5 / CSS3 / JavaScript (ES6+)
- Chart.js（グラフ描画）
- LocalStorage API（データ保存）
